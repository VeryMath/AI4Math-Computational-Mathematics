# 通用全域PDE强形式全自动转变分弱形式
## 功能说明
- 统一接口：`variation.py` 一键完成从PDE强形式到有限元标准弱形式的全流程推导
- 支持自定义输入PDE强形式残差表达式
- 自动识别二阶偏导数，完成**格林分部积分**降阶化简
- 自动拆分：体积项、本质边界项、自然边界项
- 精准分离有限元标准格式：双线性形式$a(u,v)$、线性形式$L(v)$
- 兼容常见简写语法：`u_xx`/`u_yy`/`div(grad(u))`

## 核心必执行流程
1. 残差加权构建：对PDE强形式引入检验函数，建立全域加权残差归零条件
2. 分部积分降阶：通过格林恒等式对二阶微分算子执行分部积分，降低主场变量微分阶次
3. 边界项分离：自动拆分内部体积积分项与边界轮廓积分项，区分两类边界贡献
4. 边界条件代入：将本质边界条件、自然边界条件分别代入对应边界项，消去冗余未知量
5. 标准格式整理：统一整理为有限元标准双线性形式+线性形式的弱形式表达式

## 标准输出步骤
1. 录入PDE强形式方程、全域定义、边界分区条件与对应边界条件
2. 构造加权残差积分表达式，确定试探函数与检验函数空间
3. 执行分部积分运算，完成微分算子转移与降阶处理
4. 提取并分类边界积分项，区分本质边界与自然边界贡献
5. 代入边界条件化简表达式，消去非独立未知量
6. 仅输出有限元标准弱形式最终结果：$a(u,v)=L(v)$

## 输出格式硬性要求
1. 仅输出标准弱形式的最终数学表达式，不展示推导过程
2. 采用标准可视化公式展示双线性形式$a(u,v)$与线性形式$L(v)$
3. 公式排版规整，微分算子、积分符号、边界条件符号统一规范
4. 不附加文字描述、不输出LaTeX源码，仅呈现可读数学形式
