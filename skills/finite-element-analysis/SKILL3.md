# 一键式通用有限元求解器
## 功能说明
- 若仓库或项目提供 `solve.py`、FEniCS、scikit-fem 或等价本地求解脚本，可用作完整有限元流水线入口；否则按本 Skill 的流程生成可审查的矩阵组装、边界条件和求解步骤。
- 内置三种模式：
  1. `1d_bar`：一维杆轴向拉伸/压缩
  2. `2d_scalar`：二维泊松/热传导标量场
  3. `2d_plane_stress`：二维平面应力弹性力学（适配你当前题目）
- 自动处理：单元刚度矩阵、等效节点力、边界条件、稀疏求解、后处理

## 核心必执行流程
1. 局部单元矩阵全局组装：依据单元自由度映射关系，将所有单元局部刚度矩阵、单元载荷矩阵按节点编号对应叠加，生成整体系统矩阵与总载荷向量
2. 强加边界条件统一处理：区分本质边界与自然边界，采用规范修正方式完成位移/场量约束施加，同步修正载荷向量，消除刚体运动自由度
3. 调用底层线性求解器：自动适配稀疏矩阵存储格式，调用专用线性求解模块完成方程组高效求解，输出全域未知场量

## 标准输出步骤
1. 读取网格模型、材料参数、荷载参数与边界配置信息
2. 遍历所有计算单元，批量生成局部单元矩阵与单元等效载荷
3. 完成局部矩阵向整体矩阵的精准映射与累加组装
4. 分类施加两类边界条件，完成系统方程修正
5. 对接底层线性求解器完成数值求解
6. 基于求解结果完成场量换算、数据整理与可视化输出

## 常见问题及解决方案
| 出现问题 | 对应解决方式 |
| ---- | ---- |
| 整体矩阵组装错位 | 核对单元节点自由度编号映射规则，修正索引对应关系 |
| 强加边界施加失效 | 重新校验约束节点与约束自由度，更换标准边界修正算法 |
| 线性求解结果发散 | 检查是否存在欠约束，补充边界条件或优化矩阵条件数 |
| 单元载荷叠加异常 | 统一等效载荷换算规则，规范面力、集中力转换逻辑 |

## 输出格式硬性要求
1. 矩阵相关数据、求解结果均采用规整表格与标准数学表达式展示，排版整齐有序
2. 边界条件施加过程分步呈现，明确标注约束位置、约束数值与修正方式
3. 求解器调用状态、迭代信息、求解残差按固定格式统一输出
4. 最终位移、应力、温度等结果分类汇总，区分节点结果与单元结果

## 严格禁止条例
1. 禁止随意更改单元矩阵组装映射规则，不得简化自由度匹配逻辑
2. 禁止混用不同类型边界条件施加方法，不可省略边界修正关键步骤
3. 禁止跳过正规线性求解流程，不允许手动估算、近似替代求解运算
4. 禁止删减等效节点力换算、矩阵稀疏化预处理等必要前置流程
5. 禁止擅自删减求解校验步骤，不得忽略结果合理性基础判定
6. 禁止输出编程代码、仅保留纯数学结果
