# Source Notes

These sources define the practical scope of the route families. Use official documentation first when a task requires exact API behavior.

| Source | Scope to cite |
| --- | --- |
| GUDHI documentation, https://gudhi.inria.fr/doc/latest/ | simplex trees, cubical complexes, alpha/Cech/Rips/witness complexes, persistent homology and related distances |
| Ripser.py documentation, https://ripser.scikit-tda.org/ | efficient Vietoris-Rips persistent homology, sparse/dense distance inputs, representative cocycles |
| SageMath reference, https://doc.sagemath.org/ | simplicial complexes, exact algebra, polynomial ideals, and interfaces |
| GAP HAP documentation, https://gap-packages.github.io/hap/doc/chap0_mj.html | homological algebra programming, group cohomology, CW complexes, resolutions |
| SnapPy documentation, https://snappy.computop.org/ | topology and geometry of 3-manifolds, triangulations, hyperbolic invariants, verification notes |
| Spherogram documentation, https://snappy.computop.org/spherogram.html | knots and links, PD/DT/braid/Gauss encodings, link invariants in the SnapPy ecosystem |
| Regina documentation, https://regina-normal.github.io/ | 3- and 4-manifold triangulations, normal surfaces, angle structures, knots and links |
| Macaulay2 documentation, https://macaulay2.com/ | commutative algebra, algebraic geometry, Groebner bases, resolutions, Betti numbers, Hilbert data |
| Singular documentation, https://www.singular.uni-kl.de/ | polynomial computations, standard bases, commutative algebra, singularity theory |
| OSCAR documentation, https://docs.oscar-system.org/ | computational algebra, geometry, number theory, schemes, toric geometry, homological algebra |
| Normaliz documentation, https://normaliz.uni-osnabrueck.de/ | affine monoids, rational cones, Hilbert bases, Hilbert series, lattice-point computations |

When a source is unavailable locally, do not invent exact API calls. State the intended route and ask before installing or consulting remote documentation if the task needs live API details.
