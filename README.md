# AI4Math Computational Mathematics

AI4Math Computational Mathematics collects Skill-as-adapter packages for
numerical, symbolic, and computational evidence workflows in mathematical
research.

## Skill Packages

| Package | Purpose | Entry point |
| --- | --- | --- |
| `skills/finite-element-analysis/` | Finite element analysis prompts and examples. | `README.md`, `SKILL1.md`, `SKILL2.md`, `SKILL3.md` |

## Legacy Root Packages

Existing root-level packages such as `LS-Skill/` are preserved as legacy
material. New imports should use the `skills/<skill-name>/` layout.

## Usage

Open the package you need, read its README and Skill files, then ask your coding
agent to follow that package's workflow. Keep long examples, scripts, and
references inside the package directory.
